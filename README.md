# Helldiving Wingman

An application you can run on your second monitor while playing.

Intended to provide vital information for spreading democracy.

```sh
#!/usr/bin/env bash

# https://pnpm.io/installation
curl -fsSL https://get.pnpm.io/install.sh | sh -

cd src
pnpm install
pnpm dev
```
